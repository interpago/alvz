# Lección 1: Hola Alvz y Variables

¡Bienvenido al curso oficial de Alvz! En esta primera lección aprenderás los cimientos del lenguaje.

## 1. El Comando Imprimir
En Alvz, para mostrar algo en pantalla usamos la función `imprimir()`.

```alvz
imprimir("¡Hola, futuro programador de Alvz!")
```

## 2. Variables
Usamos la palabra clave `variable` para guardar información.

```alvz
variable nombre = "Alvz"
variable version = 0.95
variable es_potente = verdadero

imprimir("Lenguaje: " + nombre)
imprimir("Versión: " + version)
```

## 3. Tipos de Datos
Alvz maneja automáticamente los tipos:
- **Texto**: Entre comillas `" "` o `' '`.
- **Número**: Enteros o decimales.
- **Booleano**: `verdadero` o `falso`.
